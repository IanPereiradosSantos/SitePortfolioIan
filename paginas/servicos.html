Serviços

HTML5

Este site utiliza corretamente a semântica adicionada ao HTML5, o que melhora a organização do conteúdo e a acessibilidade. Abaixo estão algumas das principais tags HTML5 utilizadas:

  • <header>: Define o cabeçalho da página ou de uma seção. Inclui elementos como o logotipo e o menu de navegação.
  • <nav>: Define uma seção de navegação, contendo links para diferentes partes do site.
  • <main>: Representa o conteúdo principal da página, excluindo cabeçalhos, rodapés e barras laterais.
  • <footer>: Define o rodapé da página ou de uma seção, contendo informações de copyright e links de contato.
  • <figure>: Utilizada para marcar conteúdo ilustrativo, como o logotipo, podendo incluir uma legenda com <figcaption>.

CSS3

O design do site utiliza várias propriedades e técnicas do CSS3 para aprimorar a aparência e a funcionalidade:

  • Flexbox: Utilizado em .header-container para alinhamento e distribuição flexível de elementos.
  • Media Queries: Adaptam o layout para diferentes tamanhos de tela, garantindo que o site seja responsivo. Exemplos estão em #responsivo.css.
  • Transições e Hover Effects: Adicionados para melhorar a interatividade, como o efeito hover nos links e botões.

JavaScript

O JavaScript é utilizado para manipular a apresentação dos dados do formulário após o envio. O código realiza as seguintes ações:

  • Prevenção do Comportamento Padrão: O código previne o comportamento padrão do envio do formulário para que a página não seja recarregada.
  • Coleta de Dados: Após o envio, os valores dos campos do formulário são coletados.
  • Exibição dos Dados: Os dados do formulário são exibidos dinamicamente na seção de resultados abaixo do formulário.
  • Limpeza do Formulário: Após a exibição dos dados, o formulário é limpo automaticamente para permitir novas entradas.
